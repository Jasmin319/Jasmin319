# Hi, I’m Jasmin 👋

I work at the intersection of data, structure, and decision-making, with a strong focus on Power BI, AI, and data-driven solutions.

As a Microsoft MVP and freelance BI consultant, I help turn raw data into structured, usable, and trustworthy insights — combining classic BI approaches with modern AI capabilities using Copilot and Copilot Studio.

## What I focus on

- building clean and scalable Power BI models  
- designing clear and effective KPI and dashboard layouts  
- improving data quality and data understanding  
- creating reusable BI patterns and workflows  
- integrating AI (Copilot & Copilot Studio) into BI processes
